# 9_Book Management System

## Project Overview
A simple ASP.NET Core MVC application to manage books with Database.

## Features
- Add Book
- View Book List
- Entity Framework Core
- SQL Server Database

## Technologies Used
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Bootstrap
